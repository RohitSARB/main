FastAPI Dependency Injection

FastAPI provides a powerful dependency injection system using the Depends() function.
Dependencies allow shared logic like database connections, authentication,
or configuration to be reused across multiple endpoints.

Example:

from fastapi import Depends

def get_db():
    return DatabaseConnection()

@app.get("/users")
def get_users(db = Depends(get_db)):
    return db.fetch_users()

Vector Databases

Vector databases store high-dimensional embeddings generated by models.
They allow similarity search using distance metrics like cosine similarity
or L2 distance.

FAISS is a popular open-source vector database developed by Facebook AI.

Retrieval Augmented Generation (RAG)

RAG combines retrieval systems with large language models.
First, relevant documents are retrieved.
Then, the LLM generates an answer using retrieved context.

This improves factual accuracy and reduces hallucinations.
